# Expense Track

**Expense Track** is a Flutter mobile application designed to help users manage their personal finances efficiently. Users can add, edit, and categorize their daily expenses, view summaries, and track spending patterns over time. With **Firebase** as the backend and **Provider** for state management, the app ensures real-time updates, secure authentication, and a smooth, reactive user experience.

## Key Features
- Add, edit, and delete expenses
- Categorize expenses (e.g., Food, Travel, Shopping)
- View daily, weekly, and monthly summaries
- Secure authentication (Email & Google Sign-In)
- Real-time data syncing with Firebase
- Clean and responsive UI

## Tech Stack
- **Frontend:** Flutter
- **State Management:** Provider
- **Backend & Database:** Firebase (Firestore)
- **Authentication:** Firebase Auth (Email & Google Sign-In)
- **Other Firebase Services:** Firebase Analytics, Cloud Messaging (optional)
- **Development Tools:** Android Studio / VS Code, GitHub
